# HostelXpert Mobile Application

The **HostelXpert Mobile Application** is designed to streamline daily hostel operations through a mobile-first interface. It connects directly with the HostelXpert SaaS backend, enabling administrators, staff, and operational teams to manage hostel activities efficiently from anywhere.

The application provides real-time access to essential operational features including **student management, fee tracking, mess bookings, maintenance requests, and gate pass management**.

⚠️ **Note**

The production source code for this mobile application is maintained in a **private company repository** and cannot be shared publicly due to company confidentiality policies.

This repository showcases the **application architecture, core features, and my development contributions**.

---

# Application Overview

The HostelXpert Mobile App provides operational staff with a **mobile control interface** for managing hostel activities in real time.

It enables users to perform important administrative tasks without accessing the full web panel, making daily hostel management more efficient.

Through secure API communication with the backend system, the app supports operations such as:

* Student management
* Fee payment & monitoring
* Gate pass issuing & Leave request requesting
* Mess booking and verification
* Maintenance request tracking
* Operational alerts

# Application Features

## Student Management

* View student profiles
* Monitor room allocations
* Track student hostel records

## Fee Payments

* Track student fee payments
* View payment history
* Monitor hostel fee collections

## Fee Pending Alerts

* Notify administrators about pending student fees
* Monitor overdue payments

## Gate Pass Issuing

* Issue digital gate passes for students
* Track student entry and exit permissions
* Maintain gate pass records

## Onboard Requests

* Submit onboarding requests for new students
* Manage hostel admission workflows

## Mess Bookings

* Students can book meals through the application
* Track mess participation and bookings

## Mess QR Pass

* Generate QR-based mess passes
* Verify meal collection quickly using QR validation
* Reduce manual verification during meal service

## Maintenance Requests

* Raise hostel maintenance issues
* Track complaint resolution status
* Assign maintenance tasks to staff

## Complaint Tracking

* Submit operational complaints
* Monitor complaint resolution progress

## Operational Monitoring

* Access key hostel operational information
* Monitor activity updates in real time
